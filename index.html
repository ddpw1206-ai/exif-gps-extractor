JPG to EXIF Geo Extractor

이미지의 메타데이터에서 GPS 좌표를 추출하고 도로명/지번 주소로 변환합니다.

반드시 촬영시 GPS 값을 포함한 사진이어야 하며, GPS 값이 포함되지 않은 사진은 변환되지 않습니다.

아이폰 설정 방법: 설정 앱 → 개인정보 보호 및 보안 → 위치 서비스 ON

갤럭시 설정 방법: 설정 → 위치 → ON, 카메라 앱 실행 → "위치 태그" 또는 "GPS 태그" 옵션 활성화

이미지 파일을 이곳으로 드래그 앤 드롭 하세요
또는 버튼을 클릭하여 선택 (다중 선택 가능)
